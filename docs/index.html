Installation & Setup

  • No additional dependencies required
  • All necessary components are bundled
Installer Method
  1. Download the latest .exe installer from the Releases page
  2. Run the installer and follow on-screen instructions
  3. The installer will automatically place files and create shortcuts

Portable Method
  1. Download the .zip portable version
  2. Extract to your desired location
  3. Run Mica4U.exe directly
Download Portable
Build from Source
Prerequisites:
  • Python 3.12+ (python.org)
  • pip (comes with Python)
  • PyQt6 and PyInstaller
Build Steps:
# Clone the repository
git clone https://github.com/DRKCTRLDEV/Mica4U.git
cd Mica4U

# Install dependencies
pip install -r requirements.txt

# Build the executable
./build.ps1

# Run for development
python main.py
